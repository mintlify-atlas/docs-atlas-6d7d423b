## What are stores?

Stores are storage backends that persist data between node executions in Pupload. They provide a consistent interface for uploading, downloading, and managing objects regardless of the underlying storage system.

Every flow must define at least one store to handle input and output data.

## Store interface

All stores implement a common interface:

```go
type Store interface {
    PutURL(ctx context.Context, objectName string, expires time.Duration) (*url.URL, error)
    GetURL(ctx context.Context, objectName string, expires time.Duration) (*url.URL, error)
    DeleteObject(ctx context.Context, objectName string) error
    Exists(objectName string) bool
}
```

<Note>

The local S3 implementation uses an in-memory fake S3 server:

    ts := httptest.NewServer(faker.Server())

    u, err := url.Parse(ts.URL)
    if err != nil {
        return nil, err
    }

    // ... creates minio client ...

    minioClient.MakeBucket(context.TODO(), input.BucketName, minio.MakeBucketOptions{})

    return &LocalS3Store{
        server: ts,
        client: minioClient,
        bucket: input.BucketName,

}

func NewLocalS3Store(input FilesystemS3StoreInput) (*FilesystemS3Store, error) {
    fs := afero.NewOsFs()
    fs = afero.NewBasePathFs(fs, input.MountPath)
    metaFS := afero.NewOsFs()
    metaFS = afero.NewBasePathFs(metaFS, path.Join(input.MountPath, "meta"))

    backend, err := s3afero.SingleBucket(input.BucketName, fs, metaFS)

        return nil, err

## Store operations

### Generating presigned URLs

Stores generate time-limited presigned URLs for secure data transfer:

```go
func (s *S3Store) PutURL(ctx context.Context, objectName string, expires time.Duration) (u *url.URL, err error) {
    u, err = s.client.PresignedPutObject(ctx, s.bucket, objectName, expires)
    if err != nil {
        return nil, fmt.Errorf("s3 store at %q (bucket %q): %w", s.client.EndpointURL().Host, s.bucket, err)
    }
    return u, nil
}

func (s *S3Store) GetURL(ctx context.Context, objectName string, expires time.Duration) (u *url.URL, err error) {
    u, err = s.client.PresignedGetObject(ctx, s.bucket, objectName, expires, url.Values{})
    if err != nil {
        return nil, fmt.Errorf("s3 store at %q (bucket %q): %w", s.client.EndpointURL().Host, s.bucket, err)
    }
    return u, nil
}
```

### Checking object existence

Stores can check if objects exist without downloading them:

```go
func (s *S3Store) Exists(objectName string) bool {
    _, err := s.client.StatObject(context.TODO(), s.bucket, objectName, minio.GetObjectOptions{})
    if err != nil {
        return false
    }

    return true
}
```

From `internal/stores/s3/s3.go:116-123`

This is used by the flow runtime to check if uploaded data is ready:

```go
func (rt *RuntimeFlow) checkWaitingURL(w models.WaitingURL) WaitingURLResult {
    store, ok := rt.stores[w.Artifact.StoreName]
    if !ok {
        return WaitFailed
    }
    exists := store.Exists(w.Artifact.ObjectName)
    if exists {
        return WaitReady
    }

    return WaitNoChange
}
```

## Store initialization

Stores are unmarshaled from flow configuration:

```go
func UnmarshalStore(input models.StoreInput) (models.Store, error) {
    switch input.Type {

    case "s3":
        var params s3.S3StoreInput
        if err := decodeParams(input.Params, &params); err != nil {
            return nil, fmt.Errorf("error decoding params for s3local, %w", err)
        }

        store, err := s3.NewS3Store(params)
        if err != nil {
            return nil, fmt.Errorf("Unable to create s3 store: %w", err)
        }

        return store, nil

    default:
        return nil, fmt.Errorf("invalid store tpye")
    }
}
```

<Accordion title="What happens if store initialization fails?">
If a store fails to initialize, the controller logs a warning and continues without that store. Flows referencing the failed store will error when they try to use it.

```go
func (rt *RuntimeFlow) constructStores() {
    for _, storeInput := range rt.Flow.Stores {
        store, err := stores.UnmarshalStore(storeInput)
        if err != nil {
            rt.log.Warn("Invalid store definition", "flow", rt.Flow.Name, "store", storeInput.Name, "error", err.Error())
            continue
        }

        rt.stores[storeInput.Name] = store
    }
}
```
